# ML---SmartCart-Clustering-System-
## 📊 SmartCart Customer Segmentation

SmartCart is a growing e-commerce platform with 2240 customer records and 22 features, including demographics, purchase behavior, and website activity.

Currently, the company uses generic marketing strategies, leading to inefficient campaigns, missed high-value customers, and delayed churn detection.

## 🚀 Objective
Build an intelligent customer segmentation system using unsupervised machine learning to identify hidden patterns and enable data-driven marketing decisions.

## 🧠 Approach
- Data Preprocessing & Feature Engineering  
- PCA (Dimensionality Reduction)  
- K-Means Clustering  
- Elbow Method & Silhouette Score  

## 📈 Outcome
- Identify high-value customers  
- Detect churn-prone users  
- Improve personalized marketing & retention  

## 🛠️ Tech Stack
Python, Pandas, NumPy, Scikit-learn, Matplotlib
